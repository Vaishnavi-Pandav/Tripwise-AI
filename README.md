# TripWise AI ✈️

An AI-powered travel planner that helps users:

- Generate travel itineraries
- Optimize travel budgets
- Discover attractions and hidden gems
- Compare destinations
- Get AI-powered travel assistance

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- FastAPI
- OpenAI API
- PostgreSQL

## Features

- Smart Travel Planning
- AI Itinerary Generator
- Budget Optimizer
- Hotel Recommendations
- Weather Dashboard
- PDF Export
